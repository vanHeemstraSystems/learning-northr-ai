# Extensions

## Northr.ai Google Chrome Extension

<img src="./northr-ai-google-chrome-extension.png">

This extension will (after sign-in to northr.ai) keep your Identity information available, such as business, personal, and my voice.

The extension can be installed from https://chromewebstore.google.com/detail/northr-identity-%E2%80%94-ai-iden/mimlnbciccoajmoendcpbgmjhnngooje?utm_source=item-share-cb
